end()의 핵심 동작 원리 개요

std::end()는 컨테이너의 유효한 데이터가 끝났음을 알리는 '경계선(Past-the-end)' 역할을 합니다.

만약 end()가 '마지막 데이터'를 가리킨다면, 루프를 돌 때 마지막 데이터를 처리하기 위해 코드가 매우 지저분해집니다. 

하지만 '마지막의 다음 빈 공간'을 가리키게 만들면 while (begin != end)라는 아주 깔끔한 공식 하나로 모든 데이터 구조를 순회할 수 있게 됩니다.

\
\
\

자료구조에 따라 end()가 가리키는 대상은 다음과 같이 다릅니다.

배열 기반 (예: std::vector) : 메모리가 연속되어 있으므로, 시작 주소 + 데이터의 개수를 계산하여 배열이 끝난 직후의 메모리 주소를 반환합니다.

노드 기반 (예: std::list) : 다음 칸이라는 물리적 메모리 개념이 없으므로, 기차의 끝을 의미하는 NULL 포인터를 반환합니다.
