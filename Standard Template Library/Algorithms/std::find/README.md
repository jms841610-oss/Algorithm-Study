std::find의 동작 원리 

C++의 std::find는 매우 단순하면서도 강력한 선형 탐색(Linear Search) 알고리즘입니다. 

내부적으로 복잡한 트리나 해시를 쓰지 않고, 데이터의 처음부터 끝까지 하나씩 확인하며 원하는 값을 찾습니다.

이 함수가 C언어의 일반적인 배열 탐색 함수와 구별되는 가장 큰 특징은 반개구간(Half-open interval)과 반복자(Iterator)의 개념을 사용한다는 점입니다.

시작점과 끝점 ([first, last)): 탐색을 시작할 첫 번째 위치(first)와, 탐색을 종료할 '마지막 데이터의 바로 다음 위치'(last)를 포인터로 받습니다.

순차 탐색: first 포인터를 한 칸씩 이동시키며 값을 비교합니다.

반환값: 값을 찾으면 그 위치의 포인터를 반환하고, 끝까지 못 찾으면 last 포인터를 그대로 반환하여 "못 찾았다"는 것을 알립니다. 

(C언어에서 NULL을 반환하는 것과는 조금 다른 철학이죠!)
