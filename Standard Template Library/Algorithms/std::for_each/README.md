std::for_each의 핵심 동작 원리 개요

std::for_each는 특정 범위(first, last) 안에 있는 모든 요소를 하나씩 순회하면서, 

사용자가 지정한 함수나 동작(Action)을 빠짐없이 실행해 주는 알고리즘입니다.

\
\
\

이 알고리즘의 가장 위대한 점은 **"반복하는 작업(Traversing)"**과 **"무엇을 할 것인가(Action)"**를 완벽하게 분리했다는 것입니다.

C++에서는 이를 위해 주로 람다(Lambda)나 함수 객체(Functor)를 사용하지만, 

C언어에서는 **'함수 포인터(Function Pointer)'**라는 아주 강력한 무기를 사용해서 이 마법을 구현합니다.

\
\
\

시작과 끝 (first, last): std::find와 마찬가지로 포인터를 통해 데이터의 범위를 지정합니다.

함수 포인터 전달: 각 데이터에 어떤 작업을 할지 정의한 함수의 메모리 주소를 매개변수로 넘겨받습니다.

순차적 실행: 루프를 돌면서 각 요소의 주소를 넘겨받은 함수에 하나씩 넣어줍니다.
