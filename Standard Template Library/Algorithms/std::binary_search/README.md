std::binary_search의 동작 원리 

이진 탐색의 원리는 우리가 일상에서 하는 '업다운(Up/Down) 게임'이나 '사전에서 단어 찾기'와 완벽하게 똑같습니다.

반 토막 내기: 

탐색 범위의 정중앙에 있는 값을 확인합니다.

비교와 판단:

중앙의 값이 내가 찾는 값이면? 탐색 성공!

내가 찾는 값보다 중앙의 값이 작다면? 정렬되어 있으므로 중앙 기준 왼쪽은 아예 볼 필요도 없습니다. 오른쪽 절반만 다시 탐색합니다.

내가 찾는 값보다 중앙의 값이 크다면? 마찬가지로 오른쪽은 버리고 왼쪽 절반만 탐색합니다.

반환값: 

C++의 std::binary_search는 위치를 반환하는 것이 아니라, 단순히 **그 값이 존재하는지 여부(true / false)**만 알려주는 역할을 합니다.
