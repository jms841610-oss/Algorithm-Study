std::map의 동작 원리

std::set이 데이터(Value) 자체를 기준으로 트리를 만들었다면, 

std::map은 데이터를 **키(Key)**와 **값(Value)**이라는 한 쌍으로 묶어서 저장합니다.

핵심 규칙은 다음과 같습니다.

정렬과 탐색의 기준은 오직 '키(Key)'입니다. (값은 그저 키에 매달려 있는 짐일 뿐입니다.)

키(Key)는 중복을 허용하지 않습니다. 하나의 자물쇠(키)에는 하나의 내용물(값)만 있어야 하니까요.

만약 이미 존재하는 키에 새로운 값을 넣으려고 하면, 기존 값을 새로운 값으로 덮어씁니다. (C++의 map[key] = value 동작 방식)
