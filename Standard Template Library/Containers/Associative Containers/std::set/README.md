std::set의 동작 원리 

std::set의 가장 큰 특징 두 가지는 1) 데이터가 중복되지 않으며, 2) 데이터가 자동으로 정렬된 상태를 유지한다는 것입니다.

이 두 가지 마법을 부리기 위해 C++은 내부적으로 배열이나 연결 리스트 대신 이진 탐색 트리(Binary Search Tree, BST)라는 구조를 사용합니다. 

특히 성능을 일정하게 유지하기 위해 스스로 가지를 치며 균형을 잡는 레드-블랙 트리(Red-Black Tree)라는 고급 트리를 사용하죠.


std::set을 구현하기 위해 관리해야 할 핵심 요소는 다음과 같습니다.

Node (노드): 실제 데이터와 함께, 자신보다 작은 값을 가진 왼쪽 자식 노드(left)와 큰 값을 가진 오른쪽 자식 노드(right)의 주소를 가집니다.

Root (루트): 트리의 가장 꼭대기(시작점)에 있는 노드입니다. 탐색은 항상 여기서부터 시작됩니다.
