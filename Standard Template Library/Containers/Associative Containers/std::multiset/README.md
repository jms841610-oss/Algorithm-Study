std::multiset의 동작 원리 

std::set에서는 새로운 값이 들어왔을 때 기존에 같은 값이 있으면 가차 없이 무시해 버렸죠? 

하지만 std::multiset은 동일한 값이 들어와도 환영하며 품어줍니다.

트리 구조에서 중복을 허용하는 방법은 아주 간단합니다.

값이 작으면 무조건 왼쪽으로 보낸다.

값이 크거나 같으면 무조건 오른쪽으로 보낸다. (왼쪽으로 보내는 규칙을 써도 상관없습니다!)

이렇게 하면 똑같은 값이 여러 번 들어와도 트리의 한쪽 가지를 타고 계속 이어지면서 저장되고, 

중위 순회(Inorder Traversal)를 했을 때 똑같은 값들이 나란히 예쁘게 정렬되어 출력됩니다. 

또한 multiset의 꽃이라고 할 수 있는, 특정 데이터가 몇 개나 들어있는지 세어주는 기능(count)도 추가해 보겠습니다.
