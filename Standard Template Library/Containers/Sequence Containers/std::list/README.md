std::list의 핵심 동작 원리 개요

std::list는 본질적으로 **이중 연결 리스트(Doubly Linked List)**입니다. 

데이터들이 메모리상에 뿔뿔이 흩어져 있지만, 각 데이터가 자신의 앞 데이터와 뒤 데이터의 위치(메모리 주소)를 기억함으로써 하나의 줄로 이어지는 구조입니다.

\
\
\

이를 구현하기 위해 관리해야 할 핵심 요소는 다음과 같습니다.

Node (노드): 실제 데이터와 함께, 이전 노드(prev)와 다음 노드(next)의 메모리 주소를 가지는 개별 컨테이너입니다.

Head & Tail: 리스트의 가장 첫 번째 노드(head)와 가장 마지막 노드(tail)의 주소를 기억합니다. 양방향으로 빠르게 접근하기 위함입니다.

Size: 현재 연결된 노드의 총 개수입니다.

\
\
\

vector나 deque와 달리 꽉 찼을 때 메모리를 통째로 재할당(Reallocation)하는 과정이 없습니다. 

데이터가 추가될 때마다 딱 그 데이터만큼의 메모리(노드)만 새로 할당받아 앞뒤 연결고리만 이어주면 됩니다.
