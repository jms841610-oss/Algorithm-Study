# Algorithm-Study

기초 자료구조 구현부터 실전 문제 풀이까지의 여정을 기록합니다.

\
\
\

Core Contents

Standard Template Library : STL의 내부 동작 원리를 이해하기 위해 C언어로 재구현한 라이브러리 모음입니다.

백준 : 알고리즘 문제 풀이 실력 향상을 위해서 꾸준히 문제를 풀고 복습합니다.

알고리즘 : C++ 표준을 활용하여 복잡도를 고려한 최적의 알고리즘을 구현했습니다.

프로그래머스 : 실전 감각 향상을 위해서 집중적으로 문제를 풀고 복습합니다.
\
\
\
