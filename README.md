# Algorithm-Study

개인적인 알고리즘 문제 풀이 실력 향상을 위해 학습 내용을 기록하고 관리하는 저장소입니다.

주로 C++을 사용하여 문제를 해결하며, 논리적인 사고 과정과 효율적인 코드 작성을 지향합니다.

README.md       저장소 소개 및 학습 현황



문제 풀이 원칙

단순히 정답을 맞히는 것에 그치지 않고, 다음과 같은 단계를 거쳐 깊이 있게 학습합니다.

1. 시간 & 공간 복잡도 분석: 문제의 제약 조건 내에서 해당 알고리즘이 왜 효율적인지 공학적으로 분석합니다.
   
2. 엣지 케이스(Edge Case) 고려: 논리적으로 맞더라도 실패할 수 있는 구체적인 반례 3가지 이상을 스스로 고민해 봅니다.
   
3.단계별 접근:
   1단계: 핵심 알고리즘 힌트 도출
   
   2단계: 의사코드(Pseudocode) 작성
   
   3단계: 기존 코드의 오류 분석 및 디버깅
