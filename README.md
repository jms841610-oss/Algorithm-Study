# Algorithm-Study

\
\
\

개요

백준 온라인 저지(BOJ)를 통해 해결한 문제들과 학습한 핵심 알고리즘 이론을 기록하는 저장소입니다.

효율적인 코드 작성과 논리적인 문제 해결 능력을 기르는 것을 목표로 합니다.

\
\
\

Tech Stack

Language: C++

IDE: Visual Studio Code 

Judge: Baekjoon Online Judge

\
\
\

Repository Structure

Algorithm|-----------|알고리즘 핵심 이론 정리 (Markdown)

Baekjoon|------------|문제 풀이 코드 (문제 번호_문제명.cpp)

README.md|--------|저장소 소개

\
\
\

Algorithm 

학습한 알고리즘의 핵심 원리와 구현 방법을 정리했습니다.

1. 자료구조: Stack, Queue, Hash Table, Heap, Segment Tree

2. 그래프 탐색: DFS, BFS, Dijkstra, Floyd-Warshall

3. 동적 계획법 (DP): Knapsack, LIS, LCS

4. 기타: Greedy, Divide and Conquer, Two Pointers, Binary Search

\
\
\

Rules & Conventions

1. 코드 작성: 가독성을 위해 변수명은 의미 있게 설정하며, 필요 시 주석을 활용합니다.

2. 이론 정리: 단순히 코드를 복사하는 것이 아니라, 왜 해당 알고리즘을 사용했는지 **'복잡도'**와 **'설계 이유'**를 포함합니다.

3. 업데이트: 매일 최소 1문제 이상 풀이 및 정리를 지향합니다.
