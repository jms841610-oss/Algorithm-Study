이진 탐색(Binary Search)

\
\
\

이진 탐색은 '업 앤 다운(Up & Down)' 숫자 맞추기 게임이나 사전에서 단어 찾기와 똑같은 원리입니다. 

책을 펼칠 때 첫 페이지부터 한 장씩 넘기지 않고, 

대충 중간을 펼친 뒤에 내가 찾는 단어가 앞쪽에 있는지 뒤쪽에 있는지 판단해서 반씩 뚝뚝 잘라가며 찾는 방식입니다.

\
\
\

작동 방식

이 알고리즘은 탐색 범위를 절반씩 좁혀나가는 것이 핵심입니다.

탐색할 범위의 **시작점(Left)**과 끝점(Right), 그리고 **중간점(Mid)**을 설정합니다.

중간점의 데이터와 내가 찾고자 하는 데이터(Target)를 비교합니다.

찾는 데이터가 중간점 데이터보다 크다면: 중간점보다 작은 쪽(왼쪽 절반)은 아예 볼 필요가 없으므로 시작점을 중간점 + 1로 옮겨서 오른쪽 절반만 탐색합니다.

찾는 데이터가 중간점 데이터보다 작다면: 끝점을 중간점 - 1로 옮겨서 왼쪽 절반만 탐색합니다.

데이터를 찾거나, 시작점이 끝점보다 커져서(탐색 범위가 없어짐) 찾지 못할 때까지 이 과정을 반복합니다.

\
\
\

가정 및 제한사항

가정: 

배열이나 리스트 안에 있는 데이터들은 반드시 오름차순 또는 내림차순으로 '정렬(Sorted)'되어 있어야 합니다. 

정렬되지 않은 데이터에는 이진 탐색을 쓸 수 없습니다.

제한사항: 

데이터를 자주 추가하거나 삭제해야 하는 상황이라면, 

매번 다시 정렬을 해줘야 하므로 오히려 효율이 떨어질 수 있습니다. 

이럴 때는 한 번 정렬해 두고 여러 번 검색하는 용도에 적합합니다.
